## JD Edwards API

### [JDEBIP](/jdedwards/jde-api/jdebip)
Export XML source or PDF Output from JD Edwards BLOB fields

### [JDEBSN](/jdedwards/jde-api/jdebsfn)
Export source code for BSFN from JD Edwards BLOB field (F98780R)
