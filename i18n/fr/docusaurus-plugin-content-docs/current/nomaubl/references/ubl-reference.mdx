# Référence UBL

L'écran liste l'ensemble des **termes métier UBL (BT)** transportés par le flux français de facturation électronique (EN 16931 / extended-ctc-fr / Factur-X). Chaque ligne indique l'identifiant BT, le libellé (accompagné, le cas échéant, d'une description et d'un exemple), le XPath canonique dans le document UBL et la cardinalité (`1..1`, `0..1`, `0..n`, `1..n`).

La barre permet de filtrer par section (en-tête, vendeur, acheteur, bénéficiaire, représentant fiscal, livraison, paiement, remises & charges, TVA, totaux, ligne de facture) ou de rechercher dans le BT, le XPath, le libellé, la description et l'exemple. Le tableau est regroupé par section et codé par couleur — le même code couleur est repris dans l'éditeur de facture de l'application.
